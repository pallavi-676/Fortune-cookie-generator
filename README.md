# 🥠 Fortune Cookie Generator (C++)

A fun and interactive **C++ program** that generates random fortune cookie messages along with lucky numbers — just like the ones you get in real life!

---

## ✨ Features

* 🎲 Random fortune messages (funny + motivational + wise)
* 🔢 Lucky numbers generator (0–50)
* ⚡ Simple and clean C++ logic
* 🎯 Beginner-friendly project

---

## 🛠️ Tech Used

* C++
* Standard Libraries:

  * `<iostream>`
  * `<cstdlib>`
  * `<ctime>`

---

## 🚀 How to Run

### 1. Compile the program

```bash
g++ fortune_cookie.cpp -o fortune
```

### 2. Run the program

```bash
./fortune
```
